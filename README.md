# Image Generation using Stable Diffusion

## Project Overview

This project demonstrates how to generate images from text prompts using the Stable Diffusion model.

The model converts natural language descriptions into realistic images using a diffusion-based generative process.

## Technologies Used

* Python
* PyTorch
* Diffusers Library
* Google Colab

## Model Used

Stable Diffusion (runwayml/stable-diffusion-v1-5)

## Features

* Generate images from text prompts
* Generate multiple images from a single prompt
* Control output using random seed for reproducibility
* Save generated images

## Example Prompt

"A futuristic city with flying cars at sunset"
